# form 表單

- [demo](https://vue-3-form.onrender.com/)

## 用 vite 做 form 表單：

- 姓名（text）
- 電話（tel）
- 日期（date picker）、時間（time）
- 性別（radio）
- 興趣（checkbox）＋可新增自定義選項
- 標籤輸入並新增
- 居住地（select）
- 年齡（number）
- email
- password（明碼與密碼切換）
- 體驗感受（range）
- 備註（textarea 3 行）
- 大頭貼（可預覽）
- submit 跟 reset

> 一定要用 HTML 規範去寫 form，然後再使用 vue 的寫法 preventDefault
